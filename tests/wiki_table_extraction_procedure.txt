Manual procedure for extracting tabular information from Wikipedia page data:

	1) Use URL for target wiki page on wikitable2csv, a webapp with source code available on GitHub (https://github.com/gambolputty/wikitable2csv)
	2) Copy CSV markup to CSVJSON, another webapp with source code available on GitHub (https://github.com/martindrapeau/csvjson-app)
	3) Take output JSON file and add to JSON ‘megadoc’ of all nuclear tests across all operations
